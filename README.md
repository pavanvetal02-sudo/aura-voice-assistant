# Aura 2.0 — Voice UI

React + Vite + Tailwind frontend for **Aura 2.0 (Jarvis Edition)**.

Talk or type to the orb; requests go to the Python backend at `http://localhost:8000`.

## Run

```bash
npm install
npm run dev
```

Requires the backend: see [../aura-backend/README.md](../aura-backend/README.md).

Set API URL via `VITE_AURA_API_URL` (see `.env.example`).
